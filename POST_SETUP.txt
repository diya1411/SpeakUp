🎭 HOW TO USE:

  1. Select a scenario (Job Interview, Friend, Stranger)
  2. Set comfort level (1=gentle, 5=realistic)
  3. Click "Start Conversation"
  4. Click microphone to record
  5. Speak naturally
  6. AI responds with text + voice
  7. Get coaching tip
  8. Repeat or end session

🚨 TROUBLESHOOTING:

  ❌ Microphone not working?
     → Check browser permissions
     → Try different browser
     → Verify OS mic access

  ❌ API key error?
     → Restart backend (npm start)
     → Double-check key is complete
     → No spaces in key

  ❌ Backend not starting?
     → Run: npm install (in backend folder)
     → Verify Node.js 16+ installed
     → Check port 5000 not in use

  ❌ Frontend not loading?
     → Hard refresh: Cmd+Shift+R (Mac) or Ctrl+Shift+R (Win)
     → Check console for errors (F12)
     → Make sure backend is running
