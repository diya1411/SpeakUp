## 🎯 Key Features Implemented

### Frontend (React + Vite)
- ✅ **Scenario Picker Screen** - Select from 3 conversational scenarios
- ✅ **Session Screen** - Real-time transcript, mic control, coaching tips
- ✅ **Responsive Design** - Mobile-friendly, clean white UI
- ✅ **Audio Recording** - Web Audio API with visual feedback
- ✅ **Audio Playback** - Automatic TTS playback
- ✅ **Comfort Dial** - 1-5 difficulty slider with descriptions

### Backend (Express + Node.js)
- ✅ **Transcription API** - Google Gemini speech-to-text
- ✅ **Response Generation** - Scenario-aware AI responses
- ✅ **Coaching Tips** - Generated by Gemini after each response
- ✅ **TTS Synthesis** - ElevenLabs text-to-speech
- ✅ **Session Storage** - Save conversation logs

### AI Integration
- ✅ **3 Scenarios with Custom Prompts:**
  - Job Interview (professional, evaluative)
  - Confronting Friend (emotional, realistic)
  - Talking to Stranger (awkward to warm)
- ✅ **Dynamic Comfort Levels** - Injected into every system prompt
- ✅ **Context-Aware Responses** - Uses full conversation history

## 🚀 Quick Start

### 1. **Get API Keys** (5 minutes)
   - **Gemini API:** https://aistudio.google.com/app/apikey
   - **ElevenLabs:** https://elevenlabs.io/ (API Key + Voice ID)

### 2. **Run Setup Script**
   ```bash
   # Mac/Linux
   chmod +x setup.sh && ./setup.sh
   
   # Windows
   setup.bat
   ```

### 3. **Add API Keys**
   - Edit `backend/.env` with your API keys

### 4. **Start Both Servers**
   ```bash
   # Terminal 1
   cd backend && npm start
   
   # Terminal 2
   cd frontend && npm run dev
   ```

### 5. **Open App**
   - Go to http://localhost:3000

## 🎭 Scenarios Overview

### 1. Job Interview
- **AI Role:** Professional senior hiring manager
- **Behavior:** Evaluative, asks follow-ups, challenges you
- **Level 1:** Encouraging and patient
- **Level 5:** Tough interviewer, points out gaps

### 2. Confronting a Friend

- **Level 1:** Very shy, reserved
- **Level 5:** Natural and comfortable but still guarded

## 🌐 Deployment

To make your app public on Replit:

1. Go to https://replit.com → Create → Node.js
2. Upload SpeakUp files
3. Add Secrets (your API keys)
4. Run: `cd backend && npm install && npm start` (Terminal 1)
5. Run: `cd frontend && npm install && npm run dev` (Terminal 2)
6. Click "Share" → Make "Public"
7. Share the URL with anyone!

## 🆘 Troubleshooting Quick Links

**Microphone not working?**
- Check browser permissions
- Verify mic is enabled in OS settings
- Try different browser

**API key error?**
- Restart backend
- Verify keys are correct
- Check no extra spaces

**Backend not starting?**
- Run `npm install` first
- Check port 5000 isn't in use
- Verify Node.js version 16+

**Frontend not loading?**
- Wait for Vite build (first time takes longer)
- Hard refresh browser (Cmd+Shift+R)
- Check browser console for errors

See README.md for more troubleshooting.
